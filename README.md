
# Excel-GPT

旨在使用自然语言的方式来操作 excel，基于 [chatgpt](https://chat.openai.com/) 开发。

# 使用方式

支持两种使用方式，cmd 和 ui

```shell
git clone git@github.com:acronymor/excel-gpt.git
cd excel-gpt
bash shell.bash cmd "worksheet"
```

```shell
git clone git@github.com:acronymor/excel-gpt.git
cd excel-gpt
bash shell.bash ui
```
